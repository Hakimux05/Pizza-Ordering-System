# Pizza Order System

A simple Python application to manage pizza orders using MySQL database, with order summary, receipt printing, and record viewing functions

---

- Input customer name and phone number
- Select pizza type and set quantity
- Automatic price calculation and total update
- Save orders into MySQL database
- View order summary before payment
- Print receipt in a separate window
- View all saved order records
- Input validation and error messages
- Data saving with commit and rollback

---

## Technologies Used

- **Python 3**
- **MySQL / phpMyAdmin**
- **Libraries:**
  - `tkinter` (for graphical interface)
  - `mysql-connector-python` (for database connection)
  - `datetime` (for date/time handling)

---

## How to Run

### 1. Install required library

Open your terminal or command prompt and run:

```bash
pip install mysql-connector-python
```
